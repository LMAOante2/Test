Add device
How to add a device?
1. Press the + button in the top right corner 2. Connect to devices wifi 3. Click Setup Device and wait for the window to show up 4. Enter Wifi Name, Wifi Password and Device Name 5. Click Add Device
Device Offline
Is The Device Offline?
1. Check if the device has power. 2. Try Reconnecting the device. 3. If none of the above worked try factory resetting the device.
Factory Reset
How to factory reset the device?
1. Hold the reset button for 5 seconds 2. Device is now in Provisioning mode
